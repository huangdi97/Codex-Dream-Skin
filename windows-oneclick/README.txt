Codex Dream Skin 一键安装包（Windows）

使用方法：
1. 先确认已经安装 Codex 桌面端，并且能正常登录打开。
2. 解压整个文件夹，不要只单独复制某一个文件。
3. 双击 Start-CodexDreamSkin-GUI.vbs。
4. 第一次使用点击“安装 / 修复”，它会自动安装并启动皮肤版 Codex。

如果图形界面打不开：
- 再双击 Start-CodexDreamSkin.cmd 使用备用菜单。

常用功能：
- 安装 / 修复：第一次使用、Codex 更新后、皮肤失效后都可以运行；如果 Codex 正在运行，会先提示关闭，完成后会自动启动皮肤版 Codex。
- 启动皮肤版 Codex：手动启动入口。
- 重启并启动：Codex 已经打开但没有皮肤时使用；会先弹出确认。
- 验证并截图：售后确认效果时使用。
- 换自己的图片：选择本地 PNG/JPG/JPEG/WEBP 图片，自动生成全窗口自适应主题并立即启动/刷新皮肤。
- 一键制作主题：输入主题名，选择控件配色、字体/字体文件、字体颜色、明暗模式、首页文字遮罩、任务背景和任务控件遮罩，预览图片后自动压缩生成主题文件夹，保存到本机主题库，立即启动/刷新皮肤，并自动打开效果截图。
- 主题库 / 导入主题：选择本机保存的主题，或导入卖家制作的 theme.json 主题文件夹，并立即启动/刷新皮肤。
- 桌宠制作向导：小白可以填写桌宠名字和描述，一键复制 AI 制作提示词并打开 Codex Pets；也可以安装 HTTPS 在线 sprite，或导入卖家提供的角色套装/本地桌宠。
- 恢复默认图片：恢复内置默认自适应主题并立即启动/刷新皮肤。
- 恢复官方外观：回到 Codex 原始外观。
- 卸载快捷方式：删除 Dream Skin 创建的快捷方式。
- 打开日志：遇到问题时，把日志发给卖家排查。

主题文件夹格式：
- 一个文件夹里放 theme.json 和图片文件；如果使用自定义字体，也可以放一个 .ttf/.otf/.woff/.woff2 字体文件。
- theme.json 里的 image 字段写图片文件名，例如 "image": "arina.jpg"。
- “一键制作主题”生成的主题文件夹会保存在 outputs\themes 下面，效果截图会保存在 outputs 下面。
- 任务背景可以选择智能、柔和背景、顶部横幅，或不显示背景图；任务控件遮罩可以单独控制顶部栏、内容区和输入区。
- 可自由调整的内容包括背景图、控件强调色、字体/字体文件、字体颜色、明暗模式、首页遮罩、任务背景和任务控件遮罩。为避免破坏 Codex 官方功能，不直接修改官方安装包或内部程序文件。
- 详细格式见 docs\theme-packages.txt。

桌宠 / 角色套装格式：
- 桌宠是 Codex/ChatGPT 桌面端自己的 Pets 功能。主题包只负责导入和引导选择，不改变 Codex 的任务能力。
- 自己做桌宠：点击“桌宠制作向导”，填写名字和描述，点击“复制 AI 制作提示词”，到 Codex 里粘贴发送。生成完成后，再用向导导入生成的桌宠文件夹。
- 官方入口：点击“打开 Pets 设置”，在 Codex 设置 > Pets 中选择 Create your own pet 或 Refresh。
- 在线安装：如果有 HTTPS 图片或 sprite sheet 链接，可以填入链接并点击“安装在线桌宠”。
- 单独桌宠文件夹需要包含 pet.json 和 spritesheet.png 或 spritesheet.webp。
- 角色套装可以这样放：theme.json、主题图片、可选字体文件，以及 pet\pet.json 和 pet\spritesheet.webp。
- sprite sheet 需要是透明 PNG 或 WebP，官方规格为 1536 x 1872，大小不超过 20 MB。
- 如果 sprite sheet 尺寸不对，程序会直接提示当前尺寸，方便卖家返工。
- 导入后打开 Codex 设置 > Pets，点击 Refresh，选择新桌宠；需要显示/隐藏时可在 Codex 输入 /pet。

重要说明：
- 这是第三方主题美化工具，不是 OpenAI 官方产品。
- 不提供 Codex 账号，不读取密码，不读取 API Key。
- 不修改 Codex 官方安装包，不替换 WindowsApps 或 app.asar。
- Codex 客户端更新后，可能需要重新运行“安装 / 修复”。
- 自定义图片的版权、肖像权、商标授权由图片提供者自行确认。

系统要求：
- Windows 版 Codex 桌面端。
- Node.js 22 或更新版本。
- PowerShell。

如果安装 / 修复失败：
1. 先关闭 Codex。
2. 重新双击 Start-CodexDreamSkin-GUI.vbs。
3. 点击“安装 / 修复”。
4. 如果仍失败，点击“打开日志”，把日志发给卖家排查。

如果启动失败：
1. 先关闭 Codex。
2. 重新双击 Start-CodexDreamSkin-GUI.vbs。
3. 点击“安装 / 修复”。
4. 点击“重启并启动”。
5. 仍失败时点击“打开日志”，把日志发给卖家排查。
