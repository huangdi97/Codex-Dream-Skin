Codex Dream Skin 闲鱼交付 SOP

售前确认：
1. 买家使用 Windows 还是 macOS。本安装包当前只做 Windows。
2. 买家是否已经安装 Codex 桌面端。
3. Codex 是否能正常打开并登录。
4. 买家是否已经安装 Node.js 22+。
5. 买家要默认主题、自己提供图片，还是指定某个人/角色的主题。

远程安装边界：
- 账号登录、系统密码、OpenAI 登录、API Key 都让买家本人操作。
- 不接触、不索要、不保存任何账号密码或密钥。
- 如需远程控制，操作前先说明每一步会做什么。

交付流程：
1. 发送 CodexDreamSkin-OneClick-Windows.zip。
2. 让买家完整解压。
3. 双击 Start-CodexDreamSkin-GUI.vbs。
4. 点击“安装 / 修复”，它会自动安装并启动皮肤版 Codex。
5. 如果 Codex 已打开，买家确认关闭/重启。
6. 点击“验证并截图”保存截图。
7. 教买家使用“恢复官方外观”。
8. 如果 GUI 无法打开，再使用 Start-CodexDreamSkin.cmd 备用入口。

主题定制流程：
1. 基础换图单：让买家提供 PNG/JPG/JPEG/WEBP，使用“换自己的图片”。系统会自动套用全窗口自适应布局。
2. 小白自助单：让买家点“一键制作主题”，输入主题名，选择控件配色、字体、字体颜色、明暗模式、首页文字遮罩和任务页面遮罩，预览图片并确认。程序会自动压缩图片、生成主题文件夹、保存到本机主题库、立即应用，并打开效果截图。
3. 指定人物/角色单：卖家也可以用“一键制作主题”先生成文件夹，再把 outputs\themes 里的主题文件夹发给买家导入。
4. 主题命名建议：用买家能认出的名字，例如 Arina Hashimoto Theme、Customer Idol Theme、Anime Blue Theme。
5. 想微调构图时，改 theme.json 的 art.focusX、art.focusY、art.safeArea、art.taskMode；控件配色主要改 palette.accent；字体和字体颜色改 typography.fontFamily、palette.text、palette.textMuted。

售后话术：
- Codex 更新后皮肤消失：重新运行“安装 / 修复”。
- 启动失败：关闭 Codex，运行 7 恢复，再运行 1 和 3。
- 想换图：点击“换自己的图片”，选择 PNG/JPG/JPEG/WEBP 图片，程序会自动刷新皮肤。
- 想自己做主题：点击“一键制作主题”，输入名称，选择控件配色、字体、字体颜色和任务页面遮罩，再选择图片。
- 想要某个人/角色主题：点击“主题库 / 导入主题”，选择卖家发的主题文件夹，程序会自动刷新皮肤。
- 不想用了：点击“恢复官方外观”，再点击“卸载快捷方式”。

商品说明建议：
Codex 桌面端第三方主题美化安装包，图形界面一键安装、一键启动、自定义图片、一键制作主题、自适应主题导入、一键恢复官方外观。非 OpenAI 官方产品，不提供账号，不索要密码或 API Key，不修改官方安装包。
