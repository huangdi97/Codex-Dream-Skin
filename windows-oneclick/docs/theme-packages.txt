Codex Dream Skin adaptive theme format

Easiest way:

  Open the one-click manager, click "Make theme", enter a theme name, then
  choose control color, font or font file, text color, appearance, home text
  mask, task background mode, task control mask, and one PNG/JPG/JPEG/WEBP
  image. The one-click maker compresses the image to a stable JPG theme asset
  and opens a verification screenshot after apply.

The generated theme folder is saved under:

  outputs\themes\<theme-id>\

A theme is one folder containing:

  theme.json
  one image file referenced by theme.json
  optional font file referenced by theme.json
  optional pet folder for a matching Codex desktop pet

Example:

  arina-theme\
    theme.json
    arina.jpg
    font.ttf
    pet\
      pet.json
      spritesheet.webp

Minimal theme.json:

{
  "schemaVersion": 1,
  "id": "arina-theme",
  "name": "Arina Theme",
  "image": "arina.jpg",
  "appearance": "auto",
  "art": {
    "focusX": 0.5,
    "focusY": 0.42,
    "safeArea": "auto",
    "taskMode": "auto",
    "taskChrome": "auto"
  },
  "palette": {
    "accent": "#d86b8d",
    "text": "#241f27",
    "textMuted": "#665b6b"
  },
  "typography": {
    "fontFamily": "\"Microsoft YaHei UI\", \"Microsoft YaHei\", sans-serif",
    "fontFile": "font.ttf"
  }
}

Rules:
- image must be a relative file name inside the same theme folder.
- supported image formats are PNG, JPG, JPEG, and WEBP.
- image files must be under 16 MB.
- appearance can be auto, light, or dark.
- focusX and focusY are 0.0 to 1.0 crop focus points.
- safeArea controls the home-page text/readability mask: auto, left, right,
  center, or none.
- taskMode controls whether the task/chat page shows the image: auto, ambient,
  banner, or off.
- taskChrome controls task-page readability surfaces: auto, all, content, top,
  bottom, or none.
- palette.accent can be any CSS hex color, such as #d86b8d.
- palette.text and palette.textMuted can override text colors.
- typography.fontFamily can override the UI font stack.
- typography.fontFile can reference a TTF, OTF, WOFF, or WOFF2 file in the same
  theme folder. Font files must be under 16 MB.

Optional role-suite pet:
- A matching desktop pet can be bundled as pet\pet.json plus
  pet\spritesheet.png or pet\spritesheet.webp.
- The GUI imports this pet automatically when an external theme folder contains
  pet\pet.json. Buyers can click "Pet maker wizard", then "Import role suite /
  local pet", and select the role-suite folder once; the GUI applies the theme
  and imports the pet together.
- The same wizard can copy an AI pet-generation prompt for Codex, open Codex
  Settings > Pets, install an HTTPS pet image/sprite URL, or open the local pet
  folder.
- Codex Pets expects a transparent PNG or WebP sprite sheet. The official app
  upload size is exactly 1536 x 1872 pixels and no larger than 20 MB.
- The one-click GUI checks this sprite-sheet size before copying the pet so bad
  packages fail with a clear dimension message.
- After import, open Codex Settings > Pets, click Refresh, then select the pet.
  Use /pet inside Codex to wake or hide it.

Minimal pet.json:

{
  "id": "arina-pet",
  "displayName": "Arina Pet",
  "description": "Imported by Codex Dream Skin.",
  "spritesheetPath": "spritesheet.webp"
}

Seller workflow for a requested person/character:
1. Use "Make theme".
2. Enter a theme name and select one image.
3. Optionally select a custom font file.
4. Find the generated folder under outputs\themes.
5. To sell a role suite, add a pet folder containing pet.json and a valid
   spritesheet.webp to the generated theme folder.
6. Send that generated folder to the buyer, or import it with the theme library.
7. If needed, adjust theme.json fields such as name, appearance,
   palette.accent, palette.text, palette.textMuted, typography.fontFamily,
   typography.fontFile, art.focusX, art.focusY, art.safeArea, art.taskMode,
   and art.taskChrome.

The old control preset format with dream-controls.css, dream-reference.png,
dream-background.png, and dream-palette.css is no longer used by this package.
